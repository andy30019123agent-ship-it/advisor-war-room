## 0. 目標與成功標準

**目標**：報告從「方向性資訊」升級成「可執行的專業投資建議」——每檔個股給得出：建議與時間框架、合理價值區間、風險報酬比、進出場/停損參考、部位金額、可計算的信心度；且所有數字有規則依據、與引擎資料強一致。

**成功標準（驗收）**：
1. 重產 2330 報告與週報後，首屏含決策卡：rating、三時間框架觀點、Bear/Base/Bull 合理價、R/R、部位金額（含零股標註）、信心 0-100。
2. 每個新數字能追溯到固定規則與輸入資料（報告揭露所用 EPS、PER 分位、ATR）。
3. narration 與引擎數據不一致時 build 直接 fail（用故意改壞的 narration 實測）。
4. 全部免費資料源；FinMind 額度策略已處理（token 或快取）。
5. 至少 4 檔實測：2330（大型）、2454（大型）、一檔金融股（PBR 路徑）、一檔中小型（資料稀疏路徑）。

## 3. 核心規則（定案）

### 3.1 合理價值區間（valuation.py）
- TTM EPS：優先 FinMind 財報；取不到用 `現價/PER` 反推並標註來源與降信心。
- Forward EPS = TTM EPS × (1+g)，g＝近 3/6/12 月營收 YoY 加權，clamp(-20%, +40%)。
- Bear/Base/Bull = Forward EPS × 個股 3-5 年 PER 25/50/75 分位；大盤紅燈時各檔倍數下修一檔。
- 金融/景氣循環股（產業別判斷）改 PBR×ROE 區間，不硬套 PER。
- 報告必揭露：所用 EPS、倍數、目前 PER 與分位。

### 3.2 決策引擎（decision_engine.py）
- rating ∈ {買進, 試單, 續抱, 觀望, 減碼}，由三燈分數＋估值分位＋市場燈合成（固定權重表）。
- R/R = (base fair value − 現價) ÷ (現價 − 停損參考)；R/R<1.5 一律不給「買進」。
- 停損參考 = max(entry−2×ATR14, 關鍵均線/近 20 日低)，夾在現價 -8%~-15% 內；標「參考防守位，接回自行判斷」。
- 進場條件兩型：回測型（支撐±0.5×ATR＋隔日收盤站回）／突破型（收盤破近 20 日高＋量>1.3×20 日均量）。
- 部位：0（R/R<1.5、資料不足、三面分歧、大盤紅燈且非防禦）／0.5（R/R 1.5-2、信心 50-65）／1（R/R>2、信心 65-80、市場非紅燈）／2（R/R>3、信心>80、三燈一致且波動低於自身 1 年中位）。輸出金額＋零股標註。
- 信心 0-100 = 資料完整度 30＋三燈一致性 30＋R/R 20＋市場 regime 20；缺資料逐項扣分。
- 三時間框架：短線（技術+籌碼權重高）／波段（主 rating）／中期（基本面+估值權重高），各附一句規則生成的依據＋對應參考價位。
- 失效條件三層：價格（防守位）、基本面（營收 YoY 轉負且連 2 月低於 6 月均）、籌碼（法人連 3 日同向賣且佔 20 日均量>15%）。

### 3.3 P1 模組
- **籌碼 v2**：外資/投信/自營分拆、買賣超佔 20 日均量比、分歧標記（外資賣投信買）。
- **財報品質分**：營收/EPS/毛利率/營益率/ROE/FCF/負債 各 0-2 分，連續改善加分；來源 FinMind financial_statement / balance_sheet / cashflow。
- **台股類股輪動**：族群等權 5/20/60 日動能＋量能擴張＋相對加權指數強弱，取代純 mapping。
- **事件日曆**：未來 7/14 天法說（接 tw-earnings-calendar 資料）/除息/營收公布/FOMC/CPI；「事件前高估值＋籌碼弱」自動降級規則。
- **戰績牆**：每次建議寫 recommendation_log.json（日期/價/rating/target/stop/R/R/各因子）；5/20/60 日自動回填報酬與先到 target 或 stop；報告顯示歷史命中率；權重校準每月一次、規則化不手調。

## 4. 錯誤處理
- 缺資料降級（修 Codex backlog ①）：任一資料源空表→該維度標「資料缺」、信心扣分、其餘照出；不再整檔失敗。
- 新股/資料稀疏：指標最少筆數門檻（MA120 需 ≥120 根，不足標「樣本不足」不進燈號）（修 backlog ②）。
- FinMind 402/額度：申請免費 token（環境變數 FINMIND_TOKEN，無 token 照舊跑）＋日線快取（data/cache/，同日不重抓）。
- 一致性檢查失敗＝build 中止並印 diff，禁止舊敘事上線。
